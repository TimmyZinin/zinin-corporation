Настройка LinkedIn + Threads

Инструкция для подключения AI-агента Юки к твоим аккаунтам

in
LinkedIn
~15 минут
1
Открой LinkedIn Developer Portal
Зайди по ссылке и авторизуйся своим обычным LinkedIn-аккаунтом.
linkedin.com/developers
2
Нажми "Create App"
Заполни:
- App name: любое (например, "Yuki SMM")
- LinkedIn Page: выбери свою страницу (или создай новую)
- App logo: загрузи любую картинку
- Прими соглашение
3
Запроси доступ "Share on LinkedIn"
Перейди во вкладку "Products" в настройках приложения.
Найди "Share on LinkedIn" и нажми "Request access".
Одобрение обычно мгновенное.
4
Скопируй Client ID и Client Secret
Вкладка "Auth" → скопируй:
- Client ID (длинная строка)
- Client Secret (нажми на глазик, чтобы увидеть)
Никому не показывай Client Secret! Это как пароль.
5
Получи Access Token
Перейди в OAuth Token Tools по ссылке ниже.
Выбери своё приложение, нажми "Request access token".
Скопируй полученный Access Token.
LinkedIn OAuth Tool
Access Token действует 60 дней. Через 60 дней нужно будет получить новый.
6
Узнай свой Person ID
Открой свой профиль LinkedIn в браузере.
В адресной строке будет: linkedin.com/in/kristina-zhukova/
Person ID = часть после /in/ (без слешей).
Или Тим найдёт его через API по твоему токену.
@
Threads
~20 минут
1
Открой Meta Developer Portal
Зайди по ссылке. Авторизуйся через Facebook-аккаунт.
developers.meta.com
2
Создай приложение
Нажми "Create App".
Выбери use case: "Access the Threads API".
Тип: Business.
Название: любое (например, "Yuki Threads").
Email: твой.
3
Добавь себя как тестера
В настройках приложения перейди: App roles → Roles.
Нажми "Add People", введи свой Instagram username.
Потом открой Threads: Settings → Account → Website permissions.
Прими приглашение тестера.
Твой аккаунт Threads должен быть публичным (не приватным)!
4
Скопируй App ID и App Secret
В настройках приложения: Settings → Basic.
Скопируй App ID и App Secret.
5
Получи Access Token
Перейди в Graph API Explorer по ссылке.
Выбери своё приложение.
Добавь permissions: threads_basic + threads_content_publish.
Нажми "Generate Access Token".
Авторизуйся через Instagram → скопируй токен.
Graph API Explorer
Это короткий токен (1 час). Тим конвертирует его в длинный (60 дней).
6
Узнай свой Threads User ID
Тим получит его автоматически через API по твоему токену.
Или: в Graph API Explorer сделай запрос GET /me → поле id.

Что прислать Тиму

LinkedIn Access Token (длинная строка, начинается с AQ...)
LinkedIn Person ID (или URL профиля — Тим найдёт ID)
Threads Access Token (из Graph API Explorer)
Threads App ID (из настроек Meta App)
Threads App Secret (из настроек Meta App)
Все токены — секретные! Присылай только в личные сообщения Тиму, не в групповые чаты.
