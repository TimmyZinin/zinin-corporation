# Zinin Corp — Гайдлайны по контенту

## Общие правила
1. Русский язык — основной
2. Без канцелярита и корпоративного буллшита
3. Конкретика важнее красоты
4. Данные из инструментов, не из головы
5. Антифабрикация: НИКОГДА не выдумывать цифры

## LinkedIn посты (Юки)
- Хук в первой строке (pattern interruption)
- Value-first: минимум 80% пользы, максимум 20% продвижения
- CTA в конце: подписка, комментарий, репост
- Длина: 1200-2000 символов
- Self-Refine: оценка по 5 критериям (Hook, Readability, Value, CTA, Platform Fit)

## Telegram (общие правила)
- Короткие сообщения (3-5 предложений MAX)
- Таблицы и буллеты вместо абзацев
- Эмодзи допустимы, но без злоупотребления
- HTML форматирование для ботов

## Отчёты (Маттиас, Мартин)
- Структурированные: заголовки, таблицы, числа
- Минимум 200 слов для полных отчётов
- Источник каждой цифры (инструмент или ввод Тима)
- Рекомендации в конце

## Бренды
1. Corporation — строгий, профессиональный, тёмно-синий
2. Сборка — энергичный, оранжевый, для экспертов
3. Крипто — tech-savvy, фиолетовый, для маркетологов
4. Personal — минималистичный, для LinkedIn
