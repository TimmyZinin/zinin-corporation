# 🏢 AI Corporation — Мульти-агентная система

> **Цель:** Автономная команда ИИ-агентов для управления сообществами и максимизации прибыли.

---

## 🎯 Что делаем СЕЙЧАС

**Разворачиваем CrewAI Studio на Railway** — веб-интерфейс с чатом для взаимодействия с командой агентов.

### Порядок действий:
1. ✅ OpenRouter ключ — есть
2. ⏳ Деплой CrewAI Studio на Railway
3. 📋 Создание Управленца (первый агент)
4. 📋 Создание Бухгалтера
5. 📋 Создание Автоматизатора
6. 🕐 Дизайнер (позже)

### ⚠️ Telegram НЕ ТРОГАЕМ
Создание ботов заблокировано. Всё взаимодействие — через веб-интерфейс CrewAI Studio.

## 🛠️ Технологический стек

### Платформа
- **CrewAI** — фреймворк мульти-агентных систем
- **CrewAI Studio** — веб-интерфейс (Streamlit)
- **Railway** — хостинг 24/7

### LLM
- **OpenRouter** — провайдер для всех моделей
- Управленец: `openrouter/anthropic/claude-sonnet-4-20250514`
- Остальные: `openrouter/anthropic/claude-3-5-haiku-latest`

### Память (embeddings)
- **Voyage AI** (бесплатно) или **OpenAI** ($0.02/1M токенов)

## 👑 Агенты

### 1. Управленец (ПЕРВЫЙ)
- **Роль:** CEO корпорации, координатор
- **Модель:** Claude Sonnet 4
- **Задачи:** стратегия, распределение задач, контроль бюджета, отчёты
- **Инструменты:** WebSearch, FileRead/Write, GoogleSheets, Delegate

### 2. Бухгалтер (ВТОРОЙ)
- **Роль:** Финансовый контроль
- **Модель:** Claude Haiku
- **Задачи:** учёт подписок, расходы на API, P&L, прогнозы
- **Инструменты:** GoogleSheets, Calculator

### 3. Автоматизатор (ТРЕТИЙ)
- **Роль:** Технические интеграции
- **Модель:** Claude Sonnet
- **Задачи:** публикации, webhook, cron, интеграции
- **Инструменты:** CodeInterpreter, FileWrite, Webhook

### 4. SMM-менеджер (МИГРАЦИЯ)
- **Статус:** Существует в Claude Code
- **План:** Перенести в CrewAI после базовой настройки

### 5. Дизайнер (ПОЗЖЕ)
- **Роль:** Визуальный контент
- **Инструменты:** DALL-E API
